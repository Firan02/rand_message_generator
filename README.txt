#Random Message Generator

##Features
###The application:
- Generates random inspirational messages

###Outputs:
- A day of the week
- An inspirational quote
- An emoji
- An ASCII art

In the format of, a day of the week + an inspirational quote + an emoji + an ASCII art 

The random message generator program is written in Javascriopt.
Every time the program runs, it generates a new, randomized output.
